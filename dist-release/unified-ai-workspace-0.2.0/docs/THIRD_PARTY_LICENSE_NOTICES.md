# Third-Party License Notices

This document records dependency license scan notices that require project owner review before a public distribution decision. It is not legal advice.

| --- | --- | --- | --- |
| `lightningcss` | MPL-2.0 review | Non-blocking under the current release policy | Requires project owner review before suppressing or changing policy. Keep the notice visible until reviewed. |

## Handling Policy

- Do not blindly suppress license notices.
- Keep review/unknown notices in release reports until a project owner records a decision.
- If the security policy changes, document the rationale and the exact package/license affected.
- Release 0.2.0 can proceed with these non-blocking notices under the current policy, provided the release notes/checklist call them out for owner review.
